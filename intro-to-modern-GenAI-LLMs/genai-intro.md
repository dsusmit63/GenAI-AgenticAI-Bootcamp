## 🤖 Artificial Intelligence (AI)

Artificial Intelligence (AI) is the field of building systems that can perform tasks that typically require human intelligence, such as:
- 🧠 Reasoning  
- 📚 Learning  
- 👁️ Perception  
- 🎯 Decision-making  

### 💡 Examples
- 🗺️ Google Maps chooses the fastest route  
- ♟️ Chess-playing computers  
- 📧 Spam email detection  

> [!NOTE]  
> AI systems can be **rule-based** or **data-driven (ML/DL)**  

---

## ⚙️ Different Ways to Achieve AI

---

### 1️⃣ Rule-Based Systems 🧾

Humans explicitly define rules.

✔️ Simple & predictable  
❌ Not scalable for complex problems  

#### 📌 Examples
- If temperature > 30 → Turn on AC  
- If user says "Hello" → Respond "Hi"  

---

### 2️⃣ Machine Learning (ML) 📊

ML allows systems to learn patterns from data instead of explicit rules.

✔️ Learns from data  
✔️ Improves over time  

#### 📌 Examples
- Spam detection using training data  
- 🎬 Netflix movie recommendations  

> [!NOTE]  
> ML Workflow:  
> **Data → Training → Model → Prediction**

---

### 3️⃣ Deep Learning (DL) 🧠⚡

Deep Learning is a subset of ML using **multi-layered neural networks**.

✔️ Handles complex patterns  
✔️ Works best with large datasets  
❗ Requires GPUs  

#### 📌 Examples
- 👤 Face recognition  
- 🚗 Self-driving cars  

---

### 🔬 Deep Learning Models

| Category | Models |
|--------|--------|
| Feedforward | ANN |
| Computer Vision | CNN |
| Sequence Models | RNN, LSTM, GRU, Transformer |
| Generative | GAN |
| Reinforcement Learning | DRL |

> ⭐ **Transformer = Backbone of GenAI & LLMs**

---

### 4️⃣ Generative AI (GenAI) ✨

Generative AI creates new content based on learned patterns.

#### 📌 Examples
- 💬 ChatGPT  
- 🎨 Image generation (DALL·E, Midjourney)  
- 💻 Code generation (Copilot)  

---

### 5️⃣ Agentic AI 🤖⚡

Agentic AI systems can:
- 🧠 Plan  
- 🎯 Decide  
- 🔧 Take actions  

They combine:
- LLMs  
- Tools  
- Memory  
- Reasoning  

#### 📌 Example
- ChatGPT → suggests travel plan  
- Agent → books flights, hotels & completes tasks  

---

## 📚 Important Terminologies in GenAI

---

### 🔹 LLM (Large Language Model)

LLMs are deep learning models based on **Transformers** trained on massive text data.

👉 They predict the next word/token → enabling human-like text generation  

---

### 🔹 Transformers 🔄

Transformers are the backbone of modern GenAI.

✔️ Understand context  
✔️ Capture relationships in data  
✔️ Replace older models like RNN/LSTM
